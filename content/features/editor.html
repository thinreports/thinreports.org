ThinReports Editor

ThinReports Editor は ThinReports で扱う帳票レイアウトを編集する専用のGUIツールです。 Editor を使うことによって、マウスやキーボードによる直感的な操作で業務システムで求められるような、 複雑な帳票レイアウトも簡単に作成することが可能です。 Since 0.7.0 より Windows 版に加えて Mac/Ubuntu 版もリリースしています。

ThinReports Editor は、主に以下の特徴を持ちます。

  • Since 0.7.0 レイアウト定義のエクスポート（仕様書の自動生成）
  • Since 0.7.0 単位mmによるデザイン（mm⇔pxの変換）
  • 標準フォントのビルトイン（IPAフォント/Courier/Times Roman/Helvetica）
  • タブによる複数レイアウトの同時編集
  • 定型用紙サイズ (A3/A4/B4/B5) と、ユーザ定義サイズの指定
  • 基本図形の描画、編集 (四角形、線形、楕円形、静的テキスト)
  • 動的/静的テキスト領域における横位置、縦位置指定
  • 動的Since 0.7.0/静的テキストにおける行間、文字間隔指定
  • 静的画像の埋め込み機能 (PNG/JPEG)
  • Generator によって動的にテキストデータを埋込むことが可能なテキストブロック機能
  • Since 0.7.0 Generator によって動的にイメージを埋込むことが可能なイメージブロック機能
  • 日付時刻書式、数値書式などの簡易書式機能
  • 自動改ページ、ページフッター、フッター機能を有するリスト機能
  • ユーザ補助機能
    • 拡大縮小機能
    • 元に戻す、やり直し機能 (Undo/Redo)
    • 吸着レイアウトガイド機能 (図形の整列に利用)
    • 図形の複数選択機能による一括属性変更、整列機能
    • レイヤー機能 (前面、最前面、背面、再背面)

Requirements

ThinReports Editor は、現時点で以下のプラットフォームに対応しています。

  • Windows XP/Vista/7
  • Since 0.7.0 Mac OS X 10.6+
  • Since 0.7.0 Linux (Ubuntu 11.04+)
